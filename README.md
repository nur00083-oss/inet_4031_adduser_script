# INET 4031 Lab 8 - User Creation Script

This project automates user creation on a Linux system using Python.

## Files
- create-users.py
- create-users2.py
- create-users.input

## Features
- Creates users
- Sets passwords
- Adds users to groups
- Supports dry-run mode

## How to Run

Dry run:
./create-users2.py < create-users.input

Real run:
sudo ./create-users2.py < create-users.input
